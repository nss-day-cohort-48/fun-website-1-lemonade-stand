HTML - Hyper Text Markup Language

Description

HyperText Markup Language (HTML) is the standard langauage used to design and create webpages. Often used in association with cascading style sheets and javascript, two other programming languages, HTML makes up most of what you see on the internet. HTML is currently on it's 5th version, and continues to improved upon to this day.
