Command Line

What is a terminal?

Terminal, sometimes referred to as console or command line, is a built-in tool that allows you to type instructions in order to navigate and modify content on your computer. With it, you can explore, change, and delete directories, files, and other apps and programs. The terminal not only accesses your local computer, but can also connect and sync to online accounts. For web developers, the most common external program that is used in tandem with terminal is gitHub. See links below for more information on what gitHub is and how to use it.

Useful terms and tips:

  • Directories: These are folders that contain other folders or files.
  • Repository: a central location where data is stored and managed.
  • Tab Auto-completion: when entering a location into your command line, after typing the first four characters, press the "tab" key to auto-complete the location.
  • cd~: a quick way to return to your home direcory.
