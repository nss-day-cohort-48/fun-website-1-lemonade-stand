Description

CSS stands for Cascading Style Sheets. It is a language designed to specify the overall appearance of webpages as well as the appearance and structure of the text and elements such as images and buttons on webpages and their layout.
Styles can be specified with CSS using internal style sheet definitions which are placed right into HTML/XHTML code or in external files.
CSS topics include creating stylesheets, associating styles with specific HTML elements, setting background & text color, styling lists, setting the width & height of elements, media types, and more.

CSS Syntax

    Syntax basic form

  • Selector(e.g. "h1"), curly bracket, property name(e.g. "font-size"), colon, value(e.g. "1 em"), semi-colon, closing curly bracket
  • Example- h1 {font-size: 1em;}

    • Text

  • font-famly = font used, e.g. Arial
  • font-size = text size, e.g. 60px
  • color = text color, e.g. black

    • Box model

  • content = text
  • padding = space around an element's content, inside of any defined borders;
  • e.g. padding: 2 em;
  • border = sets border style for all borders, in the format: border: (solid, dashed, doted, double) (width) (color)
  • e.g border: solid 1em black
  • margin = used to create space around elements, outside of any defined borders
  • e.g. margin: 2 em;
